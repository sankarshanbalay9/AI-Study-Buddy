## 📖 Project Overview

AI Study Buddy is an intelligent educational assistant that provides personalized study plans, AI-powered academic support, quiz generation, and learning recommendations. The project demonstrates how Artificial Intelligence can improve access to quality education and contribute to **UN SDG 4 – Quality Education**.

---

## 🌍 Sustainable Development Goal

**SDG 4 – Quality Education**

This project aims to make learning more personalized, accessible, and effective through AI technologies.
